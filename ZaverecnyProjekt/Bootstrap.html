Bootstrap 3

Bootstrap 3 vyšel roku 2013, byl nadysignován tak, aby využíval flat design a mobilní přístup.Bootstrap 3 obsahuje nové pluginy. Bootstrap 3 je nejstabilnější verzí Bootstrapu a týmem je stále podporován pro kritické opravy chyb a změny dokumentace.

Bootstrap 4

Bootstrap 4 byl poprvé oznámen v roce 2014. První verze Bootstrapu 4 vyšla o rol později v roce 2015 byl ukončen vývoj Bootstrapu 3 a Bootstrap 4 byl dokončen v roce 2018
  • Zásadní přepsání kódu
  • Nahrazení Less za Sass
  • Přidání Reboot, kolekce změn CSS specifických pro jednotlivé prvky v jediném souboru na základě Normalize
  • Ukončení podpory pro Intenet Explorer 8, Intenet Explorer 9 a iOS 6
  • Podpora CSS Flexible Box
  • Přidání možností přizpůsobení navigace
  • Přidání responzivních utilit pro rozestupy a dimenzování
  • Přechod z jednotky pixelů v CSS na root ems
  • Zvětšením globální velikosti písma ze 14px na 16px pro lepší čitelnost
  • Vypuštění písma Glyphicons icon
  • Obrovské množství užitných tříd
  • Vylepšený styl formuláře, tlačítka, rozevírací nabídky, mediální objekty a třídy obrázků

Bootstrap 5

Bootstrap byl oficiálně uveden v roce 2021. Bootstrap 5 je nejnovější verze Bootstrapu. Internet Explorer 11 a nižší však nejsou podporovány a jQuery je nahrazen JavaScriptem.
  • Nové offcanvas menu
  • Kompletně ukončili podporu pro Internet Explorer
  • Přidali custom set SVG ikon
  • Přidali custom CSS vlastnosti
  • Změna dokumentace z Jekyll na Hugo
  • Vylepšené API
  • Vylepšený grid systém
  • Podpora RTL
  • Aktualizované formuláře
  • Vylepšené přizpůsobení dokumentů
