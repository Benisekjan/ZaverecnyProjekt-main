Jazyk C

Jazyk C

Jazyk C je vyšší programovací jazyk, který přesto umožňuje zapisovat programové konstrukce tak, jak je počítač skutečně zpracovává, takže výsledný kód může být velmi efektivní, a proto v minulosti nahradil nízkoúrovňový jazyk symbolických adres.

Historie

Vývoj jazyka C začal v Bellových laboratořích AT&T mezi léty 1969 a 1973. Ritchie tvrdí, že nejpřínosnější období bylo v roce 1972. Pojmenování „C“ zvolili, protože mnoho vlastností přebírali ze staršího jazyka zvaného „B“, jehož název byl zase odvozen od jazyka BCPL (ale to není jisté, neboť Thompson také vytvořil jazyk Bon na poctu své ženy Bonnie).
V roce 1973 se stal jazyk C dostatečně stabilním. Většina zdrojového kódu jádra Unixu, původně napsaného v assembleru PDP-11, byla přepsána do C. Unix tedy patří mezi první operační systémy, které byly napsané v jiném než strojovém jazyce či assembleru. Předchozí byl například systém Multics (napsaný v PL/I) a TRIPOS (napsaný v BCPL).
 

C99

Po standardizaci jazyka v roce 1989 se většina vývoje soustředila na jazyk C++. Přesto však na konci 90. let došlo k vydání dokumentu ISO 9899:1999 (obvykle nazývaný C99), který byl následně v březnu 2000 přijat i jako ANSI standard.
C99 představil několik nových vlastností, které byly mnohdy v překladačích už implementovány jako rozšíření:
  • Inline funkce
  • Proměnné mohou být deklarovány kdekoli (jako v C++), v C89 mohly být deklarovány pouze na začátku bloku
  • Několik nových datových typů, včetně long long int (nejméně 64bitový integer), bool (logický ano/ne typ) nebo typ complex určený na reprezentaci komplexních čísel.
  • Pole s nekonstantní velikostí
  • Podpora pro zakomentování jednoho řádku //, tak jako v jazycích C++ nebo BCPL
  • Nové knihovní funkce, hlavně ve formě náhrady za funkce náchylné na přetečení na zásobníku, např. snprintf()
  • Nové hlavičkové soubory, např. stdint.h
  • Variadická makra (makra C preprocesoru s proměnným počtem argumentů)
  • Klíčové slovo restrict, které v deklaraci ukazatele specifikuje, že na paměť odkazovanou tímto ukazatelem nepřistupuje žádný jiný ukazatel (např. int* restrict foo;), díky čemuž překladač může produkovat optimalizovanější kód; zajištění, aby tomu skutečně tak bylo, je na programátorovi (při nedodržení je chování programu nedefinované)

Datové typy

 
char
Velikost : 1 byte(8 bitů)
Rozsah char -128 až 127 %c
Rozsah unsignetchar 0 až 255 %c
Užití : Můžeme použít při uložení kláves do proměnné například :
char klavesa;
printf("Pro pokračování stiskněte klávesu 'Z' ");
scanf("%c",klavesa);
 
int
Velikost : 2 - 4 byte(16 - 32 bitů)
Rozsah int -32 768 až 32 767 %d
Rozsah unsignetint 0 sž 65 535 %u
Užití : Pro uložení celých čísel.
 
float
velikost : 4 byty(32 bitů)
%f
Užití : Pro uložení desetínných čísel.
 
double
velikost : 8 byty(64 bitů)
%lf
 
long

velikost : 4 byty(32 bitů)

Podmínky

if - Pokud podmínka neplatí vykoná se podmínka ve funkci else

else if - Je použita po podmínce if kdy je znovu vyhodnocena podmínka

else - náskeduje po if nebo také else if a slouží k provedení bloku kodu pokud je podmínka nepravdivá

switch - Pužívá se, když má být provedeno více bloků kodu

 

Loopy

 
do/while

"Dělej dokud"

např. přičítej číslo dokud nebude větší než 5


	int i = 0;

	do {
		printf("%d\n", i);
		i++;
	} while (i < 5);
 
for

používá se pokud víme přesně kolikrát chceme loop opakovat

příkaz je rozdělen do 3 částí 


	for (statement 1; statement 2; statement 3) {
		//blok kodu, který má být proveden
	}
 

příklad vypíše čísla od 0 do 4


	int i;

	for (i = 0; i < 5; i++) {
		printf("%d\n", i);
	}

Funkce

//main() je funkcí


	int main(){ 
		printf("Hello World!");
		return 0;
	}

Novou funkci píšeme vždy před main funkci.

	void mojeFunkce(){
		//blok kodu
	}

void - návratová hodnota

mojeFunkce - název funkce

() - kulatá závorka do ní umístíme parametry např (int z,int y)

blok kodu uzavřeme do hranatých závorek {}

		printf("Hello World");
	}

											

	int main() {

	}
